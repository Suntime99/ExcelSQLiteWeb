## v101 Release Notes（本轮改动摘要）

### 1) 关键链路闭环（所见即所得）
- 顶栏常驻信息：当前项目 / 主表 / 输出模式 + 用户模式快捷切换。
- 文件分析（SQLite 口径）一键联动到关系识别；并支持直达工作表分析。
- 关系识别结果、MTQ/MTJ 草稿、MTJ 锁定结果：自动沉淀到项目（可恢复）。

### 2) P1：功能补齐
- 元数据扫描新增批量导入策略：
  - 工作表选择规则：第一张 / 全部 / 关键字匹配
  - 主表选择策略：第一文件第一张 / 按文件名关键字 / 按工作表关键字
  - 支持“一键加入全部源文件”“加入并导入SQLite”（自动去重）
- 关系识别批量推送支持组合键：
  - 单字段关系：字段对字段
  - 组合键/复杂关系：表达式 ON（onMode=expr），并提示人工校验
- 受控模式校验增强：
  - CTE 名称识别（WITH ...）
  - 校验失败可一键关闭受控模式继续执行

### 3) P2：一致性与可用性
- 采样参数统一：顶栏“采样设置”弹窗统一配置并回填到各模块输入框。
- 导出入口文案统一：分析报告统一标注（HTML）。

- 推荐按《v101 验收清单》进行最短闭环验证。
