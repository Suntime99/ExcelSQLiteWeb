# 自包含发布（用户无需安装 .NET）

目标：发布为 **单文件 + 自包含** 的 Windows 可执行程序，用户端无需安装 .NET 运行时。

## 1. 必要前置：WebView2 Runtime
本程序使用 WebView2 承载界面，用户电脑需要安装 **Microsoft Edge WebView2 Runtime**（Win11/部分 Win10 已自带）。

若未安装，程序会：
1) 优先尝试运行程序目录内的 `WebView2RuntimeInstallerX64.exe`（或 `WebView2RuntimeInstallerX86.exe`）；
2) 若未附带安装器，则提示并打开微软官方下载页。

## 2. 发布（推荐：win-x64）
在项目目录打开 PowerShell，执行：

```powershell
.\publish_win_x64.ps1
```

输出目录（默认）：
`.\bin\Release\net8.0-windows\win-x64\publish\`

## 3. 如需 32 位（win-x86）
```powershell
.\publish_win_x64.ps1 -Runtime win-x86
```
